# Construcción de Sitios Web

Proyectos y prácticas desarrollados durante la materia Construcción de Sitios Web de la carrera de Analista de Sistemas.

---

## 🛠️ Tecnologías utilizadas

`HTML` `CSS` `JavaScript`

## 📁 Contenido del repositorio

- Portfolio personal (HTML/CSS)
- Prácticas de maquetado y diseño web
- Ejercicios de JavaScript: arrays, funciones, DOM
- Actividad integradora final

## 📚 Conceptos aplicados

- Estructura semántica HTML5
- Diseño responsivo con CSS
- Manipulación del DOM con JavaScript
- Buenas prácticas de desarrollo web
